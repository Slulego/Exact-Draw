# Exact-Draw
Create vector graphics with easily and precisely with math calculations.

Here is the list of commands:
  * VAR Name Value
  * STROKE Width
  * FILL Color
  * POINT Name X Y
  * LINE Point1 Point2
  * LINE X1 Y1 X2 Y2
  * CIRCLE Point Radius
  * CIRCLE X Y Radius
  * ARC Point1 Point2
  * ARC X1 Y1 X2 Y2 Curve
